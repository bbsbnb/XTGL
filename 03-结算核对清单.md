# 结算核对清单

> **项目结算前，逐项核对。** 缺项 = 结算时扯皮的风险项。

## 1️⃣ 合同与概览

- [ ] 施工合同（含补充协议）
- [ ] 中标通知书
- [ ] 开工报告
- [ ] 工期延期申请/批复（如果有）

## 2️⃣ 过程签证 — 所有签证单

- [ ] 签证单是否全部签认（甲方/监理签字齐全？）
- [ ] 每份签证有对应的费用依据
- [ ] 未签认的签证：是否已提交？卡在哪个环节？

## 3️⃣ 设计变更

- [ ] 所有变更通知单是否归档
- [ ] 变更图纸是否对应最新版本
- [ ] 变更费用报审是否签认
- [ ] 变更洽商记录是否完整

## 4️⃣ 施工过程记录

- [ ] 施工日志是否完整（日期连续，无断档）
- [ ] 材料进场验收记录是否齐全
- [ ] 检验批记录与施工进度是否同步
- [ ] 与甲方/监理的会议纪要是否归档

## 5️⃣ 验收记录

- [ ] 隐蔽工程验收记录是否全部签认
- [ ] 分部分项验收记录
- [ ] 竣工验收记录
- [ ] 整改通知是否都有回复

## 6️⃣ 工程量与计价

- [ ] 所有工程量确认单已签认
- [ ] 进度款申请记录与支付情况对应
- [ ] 结算书（我方版 + 审计版）
- [ ] 对账单（双方确认）

## 7️⃣ 往来函件

- [ ] 所有甲方来函已归档
- [ ] 所有我方发函已归档（含签收记录）
- [ ] 所有监理通知已归档
- [ ] 重要函件是否有书面回复

## 8️⃣ 影像资料

- [ ] 关键节点有照片记录
- [ ] 隐蔽工程覆盖前有照片
- [ ] 照片按时间排序，与日志对应

---

## 📊 缺项处理流程

```
发现缺项
    │
    ├── 能找到的 → 立即补归档
    │
    ├── 可以补签的 → 发起补签流程
    │
    ├── 确实丢失的 → 
    │   ├── 收集间接证据（照片、日志、聊天记录等）
    │   └── 评估对结算金额的影响，做好谈判预案
    │
    └── 已确认无法补救的 → 
        └── 在索引表中标注"缺失"，计算风险金额
```

---

## ✅ 结算前最终执行步骤

1. 打开索引表，筛选状态 ≠ "已签认" 的记录
2. 列出缺项清单 + 风险等级（高/中/低）
3. 高风险的提前与甲方沟通
4. 导出索引表作为**结算证据清单附件**
5. 将整个项目文件夹打包作为**结算附件电子版**
